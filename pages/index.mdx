# XManage

Your friendly server manager written in pure bash.

<Callout type="info" emoji="ℹ️">
  XManage is currently in the early stage of development, expect any bugs.
</Callout>
## Features
- Server resources management and monitoring
- Network ports management
- Full server configuration using systemd
- Backups
- API
