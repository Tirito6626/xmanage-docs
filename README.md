# XManage

Multi-functional systemd wrapper written in pure bash.

## Features

- CPU limitations
- RAM limitations
- SWAP limitations
- Disk limitations
- Allowed network ports
- Allowed directories and files
- All features of systemd configuration

## Installation

Please visit https://xmanage.tirito.de for more information.
